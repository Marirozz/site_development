What is it?

Desktop App

It is one that depends on software that is installed directly on the workstation of each person who needs to use the system. Typically, computers are connected via the Internet to a database.

Web Application

It is a special web page, which has information on which you can interact and even change. The difference with desktop applications is that it is not installed or run on your computer, but through a browser.

Web Page

It is a document that can be accessed through a browser. The information on web pages is normally static.
